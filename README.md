AFTER HARRIER - 操作マニュアル

🕹️ 基本操作

このゲームは、PC（マウス）およびスマートフォン（タッチ操作）に対応した3Dレールシューティングゲームです。

移動

画面内を ドラッグ（スワイプ） することで自機を操作します。

PC: マウスカーソルに追従します。

スマホ: 指の動きに追従します。

攻撃

フルオート連射 です。
自機は自動的に前方にショットを撃ち続けます。プレイヤーは「回避」と「照準合わせ」に集中してください。

🖥️ 画面の見方 (HUD)

上部エリア

SCORE: 現在のスコアです。

X RATE: スコア倍率です。レベルアップやコンボで上昇します。

SHIELD (赤ゲージ): 自機の耐久力です。敵の弾や体当たり、柱への衝突で減少します。0になると CRITICAL FAILURE（撃墜） となります。

中央エリア

SYNC RATE (緑ゲージ): 現在のレベルの進行度です。

時間経過で上昇し、100%になると LEVEL UP または BOSS WARNING が発生します。

危険エリア (上下の縞模様)

CEILING LIMIT (上): 上空限界です。

WARNING (下): システムメッセージや警告が表示されます。

⚠ ゲームシステム

レベルアップと回復

敵を倒しながら生存し、SYNC RATEを100%にするとレベルアップします。

レベルアップ時、またはボス戦突入時に SHIELD（耐久力）が一定量回復 します。

チェーンリアクション

編隊を組んでいる敵（V字や列で出現する敵）を全滅させると、CHAIN REACTION が発生し、ボーナススコアが入ります。

障害物（ピラー）

地面から伸びる紫色の柱は障害物です。衝突するとダメージを受けます。

TIPS: プレイヤーの弾が柱に当たると、反射して敵に当たることがあります（跳弾）。

💀 ボス戦: HEX-OBLITERATOR

3レベルごとに巨大ボス「HEX-OBLITERATOR」が出現します。

フェーズ1: シールド展開

ボスの周囲を回転する 衛星（SATELLITE） がバリアを展開しています。

本体（コア）に攻撃は効きません。まずは周囲の衛星を全て破壊してください。

衛星は時折分離して、特攻（カミカゼ）攻撃を仕掛けてきます。

フェーズ2: コア攻撃

衛星を全て破壊するとバリアが消え、SHIELD OFFLINE となります。

露出した赤いコアを集中攻撃してください。

危険攻撃: ラムアタック（体当たり）

画面左右に「WARNING」の表示が出たら、ボスが高速で体当たりをしてきます。

警告が出ている方向の 逆サイド へ素早く退避してください。

💡 攻略のヒント

回避優先: 敵弾の速度は速めです。無理に攻撃しようとせず、まずは敵弾の軌道から外れることを意識しましょう。

音を聞く: 敵の出現やボスの攻撃予兆は、効果音でも判断できます（要サウンドON）。
